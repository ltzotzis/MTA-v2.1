To run the PicoRV32 cell level thermal simulation demo, run the following commands:

$ perl run_flow.pl -xml die.xml -stack examples/PicoRV32/die.stk
$ perl run_flow.pl -runCELL -xml examples/PicoRV32/package.xml -ptrace examples/PicoRV32/big.ptrace -mode 0

on the folder where the perl script is located.
